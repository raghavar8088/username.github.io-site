📋 Step-by-Step Guide

1

Create a new Google Sheet

Open sheets.new — a blank sheet opens instantly. Name it Bharat Market Data.

2

Open Apps Script and paste the script

In your sheet: Extensions → Apps Script
Delete all existing code, paste the script from the GoogleAppsScript.js file you downloaded.
Click Save (💾), then click Run → fetchAndWritePrices.
Grant permissions when asked. Your sheet will fill with live data!

// Key parts of the script: UrlFetchApp.fetch("https://query2.finance.yahoo.com/...") // Server-side, no CORS! sheet.getRange(...).setValues([row]) // Writes to sheet ScriptApp.newTrigger("fetchAndWritePrices") // Runs every 1 min .timeBased().everyMinutes(1).create()
3

Set up the 1-minute auto trigger

In Apps Script: Triggers (⏰ icon) → + Add Trigger
Or simply run setupTrigger() function — it creates the trigger automatically.
After this, your sheet will update with live prices every minute!

4

Publish the sheet as CSV

File → Share → Publish to web
Select: LiveData sheet → Comma-separated values (.csv) → Click Publish
Copy the URL. It looks like:

https://docs.google.com/spreadsheets/d/SHEET_ID/pub?gid=0&single=true&output=csv

Also go to Share → Anyone with link → Viewer

5

Paste your Sheet ID below and connect

Your Sheet ID is the long string in the Google Sheets URL:

https://docs.google.com/spreadsheets/d/← THIS IS YOUR SHEET ID →/edit
